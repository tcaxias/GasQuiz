# GasQuiz

A browser-based quiz game built with SvelteKit, PixiJS, and TypeScript.

## Quick Start

```bash
# install dependencies
bun install

# start dev server
bun run dev

# open in browser
open http://localhost:5173
```

## Tech Stack

| Layer     | Technology               |
| --------- | ------------------------ |
| Shell     | SvelteKit + Svelte 5     |
| Rendering | PixiJS (WebGL 2D canvas) |
| Language  | TypeScript               |
| Styling   | Tailwind CSS v4          |
| Build     | Vite                     |
| Tests     | Vitest + Playwright      |

## Development

```bash
bun run dev           # dev server with hot reload
bun run build         # production build
bun run test          # unit tests
bun run test:e2e      # end-to-end tests
bun run lint          # lint
bun run format        # format code
bun run check         # type-check
```
